## Why we changed

We rebuilt *Tidy Finance with Python* around [polars](https://pola.rs/) instead of pandas, and merged the separate R and Python books into a single set of chapters with side-by-side language tabs. The motivation: faster and more memory-efficient data wrangling, closer parity between the R and Python code, and a single source of truth that is easier to keep in sync.

Because the previous edition will not be updated, we archive its full rendered output below so existing readers, bookmarks, and citations still have somewhere to land.
